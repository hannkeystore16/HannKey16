# HANNKEY16 Premium Digital Agency Website Prompt

Create a **modern, premium, elegant, and professional website** for **HANNKEY16**, a digital agency specializing in **Website Design, Web Development, AI Integration, SEO, UI/UX Design, and Digital Business Solutions**.

The overall design should convey **trust, professionalism, innovation, and premium quality**, similar to high-end international digital agencies.

---

## Design Style

Use a clean and luxurious design language with:

* Primary Colors:

  * Black `#111111`
  * White `#FFFFFF`
  * Royal Blue `#2563EB`
* Modern typography using **Inter**, **Poppins**, or **Manrope**
* Large white space
* Soft shadows
* Rounded cards (16–24px)
* Smooth gradients using black, white, and blue
* Glassmorphism only where appropriate
* Elegant hover effects
* Smooth scrolling animations
* Fade-in and slide-up animations
* Professional spacing
* Premium iconography
* High-end corporate appearance
* Fully responsive layout (Desktop, Tablet, Mobile)

Avoid generic templates and obvious AI-generated aesthetics. The website should look custom-designed by an experienced UI/UX designer.

# Deployment & Hosting Compatibility (SmarterASP.NET)

The project must be fully compatible with **SmarterASP.NET** hosting.

Requirements:

* Ready for deployment on SmarterASP.NET shared hosting.
* Use a clean folder structure (`assets`, `css`, `js`, `images`, `fonts`).
* Avoid any dependencies that require Node.js, npm, Bun, Vite, Webpack, or a build process.
* All assets must use relative paths.
* The website must run directly after uploading files via FTP or the SmarterASP.NET File Manager.
* Ensure compatibility with IIS (Internet Information Services).
* Include a default landing page (`index.html`) or provide an `index.aspx` version if using ASP.NET.
* Avoid Linux-only features.
* JavaScript must be plain ES6 without requiring bundlers.
* CSS should be compiled and production-ready.
* Include a `.htaccess` alternative when needed and provide equivalent IIS `web.config` rewrite rules.
* Make the project easy to migrate later to ASP.NET Core or other Microsoft technologies.
* Contact forms should support PHP-independent solutions or be adaptable to ASP.NET/C# backend implementations.
* Ensure all links, images, fonts, and assets load correctly under SmarterASP.NET hosting.
